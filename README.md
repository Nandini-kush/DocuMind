# DocuMind

DocuMind is an AI-powered document chatbot that allows users to upload PDFs and ask questions from the document.

## Features

- Upload PDF documents
- Semantic search using FAISS
- Retrieval-Augmented Generation (RAG)
- Question answering using FLAN-T5
- React based chatbot UI

## Tech Stack

Frontend:
React, Axios

Backend:
FastAPI, Python

AI Stack:
Sentence Transformers  
FAISS Vector Search  
FLAN-T5 LLM  
RAG Architecture
